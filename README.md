# Proxy-Scrape
Scrapes over 85k proxies! (sometimes.)


How to start:

```
git clone https://github.com/iranathan/Proxy-Scrape.git
cd Proxy-Scrape
pip install -r requirements.txt
python proxy.py
```

It will write all working proxies to a file called `good.txt`
